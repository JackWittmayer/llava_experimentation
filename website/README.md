# Jack Wittmayer's Personal Website

This is my personal website built with HTML, CSS, and Jekyll for GitHub Pages.

## Getting Started

### Local Development

To run this site locally:

```bash
# Install dependencies
bundle install

# Run the development server
bundle exec jekyll serve
```

The site will be available at `http://localhost:4000`

## Structure

- `index.html` - Main landing page
- `styles.css` - Stylesheet for the website
- `_config.yml` - Jekyll configuration
- `README.md` - This file

## Customization

Edit the following to personalize your site:

1. **index.html** - Update the hero section, about, and project cards with your information
2. **styles.css** - Customize colors, fonts, and layout
3. **_config.yml** - Update site title, description, and URL

## Deployment

This site is set up for GitHub Pages. Simply push your changes to the `main` branch and GitHub will automatically build and deploy your site to `https://JackWittmayer.github.io`
